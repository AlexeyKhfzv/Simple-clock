### Quick Start

Install Python from [python.org](https://python.org).  
To install Kivy, run:
```bash
python -m pip install --upgrade pip
pip install kivy
```
Insert full path to `launch.desktop` instead of "*ADD YOUR PATH HERE*".  

I recommend moving updated file to your Desktop.  

### Usage

* Use this app while reading, learning, practising and more.
* Instead of small default clock.
* Use it when the quick glance at the time is important.
