### Quick Start

Install Python from [python.org](https://python.org)
To install Kivy, run:
```bash
python -m pip install --upgrade pip
pip install kivy
```
Then do this:
* Download folder `code`.  
* Add full path to `launch.desktop`.
* I recommend moving updated file to your Desktop.
